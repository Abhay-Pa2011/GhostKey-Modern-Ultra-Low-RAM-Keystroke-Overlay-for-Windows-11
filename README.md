# 👻 GhostKey: Modern Windows 11 Keystroke Overlay Engine

A high-performance, ultra-low RAM keyboard visualizer built explicitly for content creators, streamers, and software tutors. GhostKey delivers beautiful, customizable screen-space keystroke overlays without impacting system performance.

## ✨ Key Features
* 🚀 **Ultra-Low Memory Footprint:** Handcrafted minimal engine designed to run flawlessly alongside heavy games and recording suites.
* 🎛️ **Modern Control Panel:** Premium UI explicitly styled for Windows 11 visual paradigms.
* 🎭 **Dual Rendering Modes:** Toggle instantly between a **Fully Invisible Backplate** (Float Text) and a **Modern Translucent Pill Box**.
* 🔒 **Interactive Click-Through:** Locks completely into place, ensuring it never steals window focus or interrupts workflows while typing.
* 🎨 **Full Typography Control:** Dynamically tweak font families, size presets, and color palettes on the fly.
